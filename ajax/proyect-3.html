CRUD Cistel

Info Proyecto:

CRUD Web creado con PHP puro, sirve para poder registrar y administrar cada servicio (soporte, venta, garantia, etc) que el cliente solicite a la empresa. También imprime un comprobante con los datos registrados por el asesor.

Detalles del Proyecto:

  • Cliente:Cistel Perú
  • Tecnologías:PHP, HTML, CSS, JS, Bootstrap
  • Industria:Informática y Telecomunicaciones
  • Fecha:Setiembre 15, 2022
