Dashboard Web Bot

Info Proyecto:

Dashboard Web que muestra los datos recogidos por el bot en una base de datos, sirve para poder mostrar las distintas estadisticas que se necesiten. También para poder cambiar algunas opciones del bot segun los requisitos de la empresa, ademas se pueden controlar todos los chats que el bot ha tenido durante ese tiempo, asi como apagar o prender el bot para distintas personas. Un asesor lo puede utilizar para enviar mensajes mediante el Bot.

Detalles del Proyecto:

  • Cliente:Tigal Group
  • Tecnologías:Laravel, HTML, CSS, JS, Bootstrap, Docker
  • Industria:Branding, Diseño y Marketing
  • Fecha:Noviembre 1, 2023
