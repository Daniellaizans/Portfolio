DASHBOARD BOT

Información del Proyecto:

Sistema Web para la administracion de los datos recolectados por un bot, este sistema mostrara de forma sencilla e intuitiva los datos recolectados por el BOT de Whatsapp, también permite que la empresa pueda modificar el bot a su misión empresarial, asi como usar el canal del BOT como medio de comunicación para con los clientes.
